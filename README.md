# Chinese Typing Game

A small TypeScript + React typing game scaffolded with Vite.

Features
- Vite React + TypeScript
- Simple typing UI with WPM & accuracy
- Local quote library persisted to localStorage
- Add your own quotes from the UI / Custom character entry

Getting started
1. npm install
2. npm run dev
3. Open http://localhost:5173

Planned improvements
- Score saving and history
- Import / export quotes in admin account
- Better UI / styling
